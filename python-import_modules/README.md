# python-import_modules

This project covers importing functions and variables between Python
files/modules, using command line arguments, and inspecting compiled
modules.

## Learning Objectives

- Why Python programming is awesome
- How to import functions from another file
- How to use imported functions
- How to create a module
- How to use the built-in function `dir()`
- How to prevent code in your script from being executed when imported
- How to use command line arguments with your Python programs

## Tasks

| File | Description |
| ---- | ----------- |
| `0-add.py` | Imports and uses `add` from `add_0.py` |
| `1-calculation.py` | Imports and uses functions from `calculator_1.py` |
| `2-args.py` | Prints the number of and list of arguments passed |
| `3-infinite_add.py` | Prints the sum of all arguments passed |
| `4-hidden_discovery.py` | Prints all public names in `hidden_4.pyc` |
| `5-variable_load.py` | Imports and prints variable `a` from `variable_load_5.py` |

## Requirements

- Ubuntu 20.04 LTS, python3 (3.8.5)
- First line of every file: `#!/usr/bin/python3`
- pycodestyle 2.7.*
- All files executable
