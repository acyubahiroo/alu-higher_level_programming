# python-classes

## Description

This project is part of the Higher Level Programming curriculum. It covers
Python classes and object-oriented programming: how to define a class, use
private attributes, getters and setters (properties), and instance methods,
using a `Square` class as a working example.

## Learning Objectives

- Why Python programming is awesome
- Why object-oriented programming is awesome
- "First-class everything"
- What's a class
- What's an object and an instance
- What's the difference between a class and an object or instance
- What's an attribute
- What are and how to use public, protected, and private attributes
- What's `self`
- What's a method
- What's the special `__init__` method and how to use it
- What's Data Abstraction, Data Encapsulation, and Information Hiding
- What's a property
- How to use the `property`, `getter`, and `setter` decorators
- What's the difference between an attribute and a property in Python
- How to dynamically create arbitrary new attributes for existing instances
  of a class
- How to bind attributes to object and classes
- What's the `__dict__` of a class and/or an instance of a class
- How does `dir()` work and how to use it

## Requirements

- Editors allowed: vi, vim, emacs
- All files are interpreted on Ubuntu 20.04 LTS using python3 (version 3.8.5)
- All files end with a new line
- The first line of all files is exactly `#!/usr/bin/python3`
- Code follows the `pycodestyle` style (version 2.7.*)
- All files are executable
- No modules may be imported unless a task explicitly allows it

## Tasks

| File | Description |
| --- | --- |
| `0-square.py` | Defines an empty `Square` class |
| `1-square.py` | `Square` with a private instance attribute `size` |
| `2-square.py` | `Square` with `size` validation (type and value) |
| `3-square.py` | `Square` with an `area()` method |
| `4-square.py` | `Square` with `size` getter/setter using `property` |
| `5-square.py` | `Square` with a `my_print()` method that prints the square |
| `6-square.py` | `Square` with a `position` property for printing the square |
